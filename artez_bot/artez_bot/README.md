# ARTEZ Telegram Bot

## Установка на Railway.app

1. Зарегистрируйтесь на railway.app
2. Нажмите "New Project" → "Deploy from GitHub repo"
   ИЛИ нажмите "New Project" → "Empty service"
3. Загрузите все файлы из этой папки
4. В настройках проекта добавьте переменные окружения:
   - BOT_TOKEN = ваш токен
   - ADMIN_ID = 624826036
   - SHEETS_URL = ваш URL Google Script
5. Нажмите Deploy

## Команды бота
/start - Выбор языка и главное меню
/order - Оставить заявку
/calc  - Калькулятор стоимости
/prices - Цены
/branches - Наши филиалы
